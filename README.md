# EMPA-Analysis-Macauley
Microprobe (EMPA) data for samples from sediment cores off of Macauley submarine caldera volcano. 
Microprobe settings - Settings used in the analysis of glass and crystal components from samples within cores IN2022_07PC and IN2022_08PC
EMPA_SEM_EDS_METHOS_Silicates_Macauley - Methods document for the EMPA analysis of glass and crystal components from samples within cores IN2022_07PC and IN2022_08PC
Silicates_all_sessions_comb - Full dataset from samples within cores IN2022_07PC and IN2022_08PC. Sample key in sheet1
